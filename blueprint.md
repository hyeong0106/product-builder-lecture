# 로또 번호 추첨기 (Lotto Number Generator)

## 개요

이 애플리케이션은 사용자가 버튼 클릭으로 무작위 로또 번호를 생성할 수 있는 현대적이고 직관적인 웹 기반 로또 번호 추첨기입니다. HTML, CSS(Variables), JavaScript(ES Modules)를 사용하여 구축되었으며, 다크 모드와 라이트 모드를 지원하여 사용자 맞춤형 경험을 제공합니다.

## 기능

- **번호 생성:** 버튼 클릭 시 1부터 45까지의 숫자 중 중복되지 않는 6개의 무작위 번호를 생성하고 오름차순으로 정렬합니다.
- **테마 전환:** 다크 모드와 라이트 모드를 자유롭게 전환할 수 있으며, 사용자의 선택은 브라우저에 저장되어 재방문 시에도 유지됩니다.
- **애니메이션:** 번호가 생성될 때 순차적으로 나타나는 페이드인 및 스케일 애니메이션을 통해 시각적 즐거움을 제공합니다.
- **번호별 색상 차별화:** 로또 번호 대역별(1-10, 11-20 등)로 고유한 색상을 부여하여 가독성을 높였습니다.
- **반응형 디자인:** 모바일 및 데스크톱 환경 모두에 최적화된 레이아웃을 제공합니다.

## 디자인 및 스타일

- **CSS 변수 활용:** `:root`와 `[data-theme="dark"]`를 사용하여 테마별 색상 팔레트를 효율적으로 관리합니다.
- **현대적인 UI:** 부드러운 그림자(Soft Shadows), 둥근 모서리(Rounded Corners), 그리고 세련된 그라데이션 배경을 적용했습니다.
- **상호작용성:** 버튼 호버 효과, 테마 토글 애니메이션 등 사용자 동작에 즉각적으로 반응하는 요소를 포함합니다.
- **타이포그래피:** 'Segoe UI' 등 시스템 폰트 스택을 사용하여 깔끔하고 가독성 좋은 텍스트를 제공합니다.

## 구현 상세 (최근 변경 사항)

- **다크 모드 지원:** CSS Variables를 도입하여 배경색, 텍스트색, 버튼 스타일 등을 테마에 따라 동적으로 변경합니다.
- **테마 저장 기능:** `localStorage`를 사용하여 사용자의 테마 설정을 브라우저 세션 간에 유지합니다.
- **테마 토글 버튼:** UI 우측 상단에 직관적인 아이콘(🌙/☀️)을 배치하여 손쉽게 모드를 전환할 수 있게 했습니다.

## 향후 계획

- **번호 저장 기능:** 생성된 번호 이력을 로컬에 저장하고 확인하는 기능 추가.
- **당첨 확인 API 연동:** 최신 로또 당첨 번호와 비교하는 기능 검토.
