# 로또 번호 추첨기 (Lotto Number Generator)

## 개요

이 애플리케이션은 사용자가 버튼 클릭으로 무작위 로또 번호를 생성할 수 있는 현대적이고 직관적인 웹 기반 로또 번호 추첨기입니다. HTML, CSS(Variables), JavaScript(ES Modules)를 사용하여 구축되었으며, 다크 모드와 라이트 모드 지원, 그리고 제휴 문의 기능을 포함하고 있습니다.

## 기능

- **번호 생성:** 버튼 클릭 시 1부터 45까지의 숫자 중 중복되지 않는 6개의 무작위 번호를 생성하고 오름차순으로 정렬합니다.
- **테마 전환:** 다크 모드와 라이트 모드를 자유롭게 전환할 수 있으며, 사용자의 선택은 브라우저에 저장되어 재방문 시에도 유지됩니다.
- **제휴 문의:** Formspree를 연동하여 이메일과 메시지를 통해 제휴 문의를 보낼 수 있는 폼을 제공합니다.
- **댓글 기능:** Disqus를 연동하여 사용자 간의 소통과 피드백을 수집할 수 있는 댓글 섹션을 하단에 배치했습니다.
- **애니메이션:** 번호가 생성될 때 순차적으로 나타나는 페이드인 및 스케일 애니메이션을 통해 시각적 즐거움을 제공합니다.
- **번호별 색상 차별화:** 로또 번호 대역별로 고유한 색상을 부여하여 가독성을 높였습니다.
- **반응형 디자인:** 모바일 및 데스크톱 환경 모두에 최적화된 레이아웃을 제공합니다.

## 디자인 및 스타일

- **CSS 변수 활용:** 테마별 색상 팔레트와 입력 필드 스타일을 효율적으로 관리합니다.
- **현대적인 UI:** 부드러운 그림자(Soft Shadows), 둥근 모서리(Rounded Corners), 세련된 그라데이션 배경을 적용했습니다.
- **상호작용성:** 버튼 호버 효과, 테마 토글, 문의 폼 열기/닫기 등 풍부한 인터랙션을 제공합니다.
- **반응형 댓글창:** 댓글 섹션은 너비가 자동으로 조절되며 모바일에서도 편리하게 사용할 수 있습니다.

## 구현 상세 (최근 변경 사항)

- **제휴 문의 폼 추가:** `Formspree`를 사용하여 백엔드 코드 없이 이메일 문의를 수집할 수 있도록 구현했습니다.
- **Disqus 댓글 시스템 통합:** 페이지 하단에 실시간 댓글 소통이 가능한 Disqus 위젯을 추가했습니다.
- **동적 UI:** '제휴 문의하기' 버튼을 통해 폼을 열고 닫을 수 있는 인터랙션을 추가했습니다.
- **테마 통합:** 문의 폼 또한 다크 모드와 라이트 모드 테마에 맞춰 스타일이 자동 조정됩니다.

## 설정 및 주의 사항

- **Disqus 설정:** `main.js` 파일 내 `loadDisqus` 함수에서 `productbuilder-7xuyaqptkc` shortname이 적용되었습니다. 필요 시 해당 파일에서 다른 shortname으로 변경 가능합니다.
- **Formspree 설정:** 제휴 문의 폼의 `action` URL을 사용자의 Formspree 엔드포인트로 변경하여 실제 문의를 받을 수 있습니다.

- **번호 저장 기능:** 생성된 번호 이력을 로컬에 저장하고 확인하는 기능 추가.
- **당첨 확인 API 연동:** 최신 로또 당첨 번호와 비교하는 기능 검토.
