# 로또 번호 추첨기

## 개요

이 애플리케이션은 사용자가 버튼 클릭으로 무작위 로또 번호를 생성할 수 있는 간단한 웹 기반 로또 번호 추첨기입니다. 이 프로젝트는 HTML, CSS, JavaScript를 사용하여 만들어졌으며 최신 웹 표준과 웹 구성 요소를 활용하여 현대적이고 상호작용적인 사용자 경험을 제공합니다.

## 기능

- **번호 생성:** 버튼 클릭 시 1부터 45까지의 숫자 중 6개의 고유한 무작위 번호를 생성합니다.
- **애니메이션:** 생성된 번호가 화면에 애니메이션과 함께 표시되어 시각적 매력을 더합니다.
- **반응형 디자인:** 애플리케이션은 데스크톱과 모바일 장치 모두에서 잘 보이도록 완전히 반응형으로 설계되었습니다.

## 디자인 및 스타일

- **미학:** 현대적인 디자인 가이드라인을 따르는 깔끔하고 직관적인 사용자 인터페이스.
- **색상 팔레트:** 생생하고 활기찬 느낌을 주는 다양한 색상 집중도와 색조.
- **타이포그래피:** 명확하고 읽기 쉬운 글꼴을 사용하여 중요한 정보를 강조합니다.
- **시각 효과:** 깊이감을 주기 위해 다층적인 그림자 효과를 사용합니다.
- **상호작용성:** 버튼과 같은 상호작용 요소는 "빛나는" 효과를 가져 사용자 참여를 유도합니다.

## 현재 계획

- **`index.html` 설정:** 로또 번호 추첨기의 기본 HTML 구조를 만듭니다.
- **`style.css` 스타일링:** 애플리케이션의 시각적 디자인과 레이아웃을 정의합니다.
- **`main.js` 기능 추가:** 로또 번호 생성 논리를 구현하고 웹 구성 요소를 사용하여 번호를 표시합니다.
